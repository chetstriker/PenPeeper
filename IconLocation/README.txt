ICON LOCATION FOLDER
====================

This folder contains 32x32 PNG icon files for device types.

HOW TO ADD ICONS:
-----------------
1. Find or create a 32x32 PNG image for the device type
2. Save it to this folder with the exact filename listed below
3. The application will automatically load it

REQUIRED ICON FILES:
--------------------
unknown.png - Default/Unknown device
windows.png - Windows (generic)
windows_98.png - Windows 98
windows_vista.png - Windows Vista
windows_me.png - Windows Millennium Edition
windows_7.png - Windows 7
windows_8.png - Windows 8
windows_8_1.png - Windows 8.1
windows_10.png - Windows 10
windows_11.png - Windows 11
windows_iot.png - Windows IoT
windows_embedded.png - Windows Embedded
windows_server_2012.png - Windows Server 2012
windows_2016.png - Windows Server 2016
windows_2019.png - Windows Server 2019
windows_2022.png - Windows Server 2022
windows_2025.png - Windows Server 2025
linux.png - Linux (generic)
unix.png - Unix
centos.png - CentOS
redhat.png - RedHat
ubuntu.png - Ubuntu
router.png - Router
firewall.png - Firewall
printer.png - Printer
pbx.png - PBX
ip_phone.png - IP Phone
radiology.png - Radiology Modality
biomed.png - Biomed
nas.png - NAS
smartphone.png - Smart Phone (generic)
iphone.png - iPhone
android.png - Android
palm.png - Palm
windows_mobile.png - Windows Mobile
iot.png - IoT Device
ip_camera.png - IP Camera
door_controller.png - Door Controller
kiosk.png - Kiosk
ups.png - UPS
vmware.png - VMware
default.png - Fallback icon (used when others fail to load)

NOTES:
------
- All icons should be 32x32 pixels
- PNG format with transparency recommended
- If an icon file is missing, the default.png will be used
- If default.png is missing, a built-in icon will be shown
- You need to update icon_list.dart with the info so the app can find them.
